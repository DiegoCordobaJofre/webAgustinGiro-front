# Assets - Imágenes

## Logo
Coloca tu logo en formato PNG en este directorio con el nombre:
- `logo.png`

El logo aparecerá en la parte superior izquierda del header. Si no hay logo, se mostrará el texto "Agustín Giro".

## Imagen Hero
Coloca la imagen principal para la página de inicio en este directorio con el nombre:
- `hero-image.jpg`

Esta imagen se mostrará como banner principal en la página de inicio. Recomendaciones:
- Resolución: 1920x1080px o superior
- Formato: JPG o PNG
- Tamaño: Optimizado para web (máximo 500KB)

## Proyectos
Las imágenes de los proyectos se gestionan desde el panel de administración (`/admin/dashboard`). Puedes subir imágenes directamente desde allí cuando el backend esté configurado.
