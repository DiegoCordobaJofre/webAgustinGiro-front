# Web Agustín Giro

Sitio web del arquitecto Agustín Giró: portfolio de proyectos, sobre mí y contacto. Incluye panel de administración para gestionar proyectos (CRUD).

- **Tecnologías:** Angular 17, TypeScript, SCSS
- **Rutas:** `/` inicio, `/proyectos`, `/sobre`, `/contacto`, `/admin` (login y dashboard)

## Cómo ejecutar

```bash
npm install
npm start
```

Abre `http://localhost:4200`. La API se configura en `src/environments/` (desarrollo y producción).
